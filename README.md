## 스터디 규칙

- 문제 파일명 : `이름.js` (예시: `구련아.js`)
- 문제풀이 설명 파일명 : `이름.md` (예시: `구련아.md`)
- 스터디 전날 오전까지 문제 모두 풀기
- 스터디 시작 전까지 문제 하나 선정해서 issue로 올리기

### 깃 컨벤션

- 문제 푼 경우 : `solve: 문제` (예시: `solve: 프로그래머스 괄호변환`)
- 문제 수정한 경우 : `refactor: 문제` (예시: `refactor: 프로그래머스 괄호변환`)
